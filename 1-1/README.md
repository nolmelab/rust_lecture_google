# 1일차 개요

강의 첫 날입니다. 오늘 배울 것이 참 많습니다:

* 러스트 기본 문법: 변수, 스칼라 / 복합 타입, 열거형, 구조체, 참조형, 함수와 메서드.
* 메모리 관리: 스택과 힙, 수동 메모리 관리, 스코프(범위)기반 메모리 관리, 가비지 컬렉션(GC)
* 소유권: 이동 의미론(move semantics), 복사와 복제, 빌림, 수명.
